## 🖥&ensp;下載專案
> clone 專案  
```
git clone https://github.com/bftsai/productFilter.git  
```

> 移至專案資料夾  
```
cd localFolder/productFilter  
```

> 安裝套件  
```
npm install  
```

> 啟動專案  
```
npm run dev  
```
